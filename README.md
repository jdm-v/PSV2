## Welcome to my personal portfolio website's repo!

I was able to get this up and running [here]("https://numberjuan.dev/") with a lot of cool and interesting tech, including but not limited to:\
    [Typescript](https://www.typescriptlang.org/docs/) + [React](https://react.dev/learn/setup) + [Astro](https://docs.astro.build/en/concepts/why-astro/) (Frontend)\
    [Github](https://github.com/jdm-v) + [Docker](https://docs.docker.com/get-started/docker-overview/) + [Netifly](https://www.netlify.com/about/) + [Porkbun](https://porkbun.com/about) (Deployment + Hosting)\
    [VSCode with Agents](https://code.visualstudio.com/docs/copilot/agents/third-party-agents) + [Open WebUI with Ollama](https://github.com/open-webui/open-webui) + [ComfyUI](https://docs.comfy.org/) (Development)
